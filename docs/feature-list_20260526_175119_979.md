## 当前版本功能（v0.4.0）

| 模块 | 功能 | 说明 |
|------|------|------|
| **场景系统** | 启动 → 菜单 → 游戏 流程 | 完整场景切换，暂停/恢复 |
| **探索系统** | 随机地图生成 + 玩家移动 + 相机跟随 | 40×30 格地图，WASD/摇杆双输入，平滑相机 |
| **实体系统** | 玩家/丧尸/幸存者 | 丧尸巡逻/追击 AI，幸存者对话，物资收集 |
| **战斗系统** | AP 回合制战斗 | 玩家 3AP：攻击/防御/撤退/等待；丧尸类型区分；战斗结算 |
| **HUD** | 生命/理智/天数/时间/资源显示 | 实时刷新，HTML 覆盖层 |
| **移动端** | 虚拟摇杆 + 交互按钮 | 左侧摇杆 35%，右侧按钮 35%，触摸事件隔离 |
| **PWA** | 横屏独立应用 | manifest.json，viewport-fit=cover，三档响应式 |
| **时间系统** | 游戏内时间推进 | 1 秒 = 1 分钟，跨天计算 |
| **暂停系统** | ESC 暂停，恢复/存档/退出 | 暂停遮罩层，事件正确解绑 |
| **通知系统** | 屏幕顶部滑入通知 | 3.5 秒自动消失，info/warning/error/success 类型 |
