# 「幸存者」美术风格设计文档

> 版本：v0.1.0 | 日期：2026-05-26

---

## 1. 整体美术定位

### 1.1 核心风格

**2D 卡通风格**为主，参考《植物大战僵尸》的视觉语言——线条简洁、造型夸张、色彩明快中带有末世灰调。

### 1.2 风格关键词

| 关键词 | 说明 |
|--------|------|
| 2D 卡通 | 人物与 UI 采用卡通渲染风格 |
| 欧式 2.5D | 地图建筑有体积感，但保持手绘质感 |
| 俯视角 | 探索阶段切换为俯视视角 |
| 末世灰调 | 整体色调偏冷灰，废墟氛围 |
| 美式漫画 | 部分过场与特效参考美漫表现手法 |

### 1.3 多视角切换方案

| 场景 | 视角 | 参考 |
|------|------|------|
| 基地建设/管理 | 欧式 2.5D 侧视 | 模拟经营类游戏 |
| 随机探索 | 俯视角 | 《这是我的战争》 |
| 对话/剧情 | 角色半身立绘 + 对话框 | AVG 视觉小说 |
| 战斗演出 | 2D 横版动画 | 卡牌 RPG 战斗 |

## 2. 角色设计方向

### 2.1 主角设计

- **造型风格**：西方家庭设计语言，贴近欧美卡通角色比例（约 3~4 头身）
- **服饰设计**：末世生存装扮，强调实用性与层次感（夹克、工装裤、背包）
- **表情系统**：涵盖疲劳、紧张、希望、决心等末世情感状态

### 2.2 幸存者设计

- **多样性**：不同年龄、性别、职业背景的幸存者
- **视觉区分**：通过服装颜色、体型、配饰区分意志强弱与属性倾向
- **状态表现**：血量低时呈现受伤动画，意志薄弱时呈现畏惧姿态

### 2.3 丧尸设计

- **普通丧尸**：腐败的市民形象，动作迟缓
- **变异体**：扭曲变形的特殊造型，颜色更鲜艳（警示性）
- **残留意识丧尸**：保留部分人类特征，眼神中留有一丝理智的痕迹

### 2.4 佣兵设计

| 等级 | 视觉特征 |
|------|----------|
| S 级 | 精致设计、独特服装、专属配色、个性化配饰 |
| A 级 | 统一风格但各有特色，标准化战斗/工作装备 |
| B 级 | 简朴统一的支援型服装，功能性为主 |

## 3. 场景美术设计

### 3.1 建筑风格

以**欧式 2.5D** 为主，建筑呈现体积感但不追求写实：

- **城市废墟**：欧式公寓楼、砖石结构、碎裂的玻璃窗
- **商铺**：复古招牌、翻倒的货架、散落的商品
- **农田**：荒废的温室、干涸的灌溉系统、杂草丛生
- **安全屋**：加固的墙壁、铁丝网、自制防御设施

### 3.2 色调体系

| 区域 | 主色调 | 情绪 |
|------|--------|------|
| 城市街道 | 灰蓝 + 锈红 | 荒废、危险 |
| 商铺/室内 | 暖黄 + 暗棕 | 物资线索、探索感 |
| 安全屋 | 暖橙 + 木色 | 安全、希望 |
| 农田/户外 | 枯黄 + 灰绿 | 荒凉、资源稀缺 |
| 特殊区域 | 高对比度 | 紧张、关键事件 |

### 3.3 光照方案

- 自然光：偏冷、漫反射，营造阴天末日的常驻氛围
- 室内光：局部暖光（蜡烛、手电筒），制造明暗对比
- 警示光：红色闪烁用于危险区域提示
- AIGC 生成立绘保持统一光源方向

## 4. UI/UX 设计方向

### 4.1 界面风格

- 扁平化 + 手绘纹理叠加，模拟末世手工制品的质感
- 图标设计参考乐高式模块化风格——简洁、可辨识
- 字体选择粗体无衬线风格，确保可读性

## 5. AIGC 立绘方案

### 5.1 应用场景

AIGC 技术用于以下内容的批量生成：

- 幸存者/NPC 角色立绘
- 佣兵角色立绘（各等级）
- 丧尸/变异体造型设计
- 道具图标的概念稿
- 场景概念图

### 5.2 风格统一策略

| 环节 | 措施 |
|------|------|
| Prompt 工程 | 建立统一的正/反向 Prompt 模板，锁定画风关键词 |
| 后处理管线 | 统一调色板映射、线稿风格化滤镜 |
| 质量控制 | 人工审核 + 风格一致性校验（色域、笔触、比例） |
| 迭代流程 | 初稿 AIGC 生成 → 风格统一后处理 → 人工精修 → 入库 |

### 5.3 推荐技术路线

| 阶段 | 工具方案 |
|------|----------|
| 概念探索 | Midjourney / Stable Diffusion 文生图 |
| 批量生成 | Stable Diffusion + LoRA 风格模型 |
| 精修定稿 | Photoshop / Clip Studio Paint |
| 资产管理 | 统一命名规范 + 版本管理 |

### 5.4 立绘规格

| 项目 | 规格 |
|------|------|
| 分辨率 | 2048 × 2048 px（角色全身） |
| 格式 | PNG（透明背景） |
| 表情变体 | 每角色 5~8 种表情 |
| 颜色模式 | sRGB |

## 6. 动画设计方向

### 6.1 动画风格

- 2D 骨骼动画为主，保持卡通弹性质感
- 关键帧手绘补充特殊效果（爆炸、血液飞溅等）

### 6.2 核心动画列表

| 角色 | 动画 |
|------|------|
| 主角 | 待机、行走、跑步、攻击（近战/远程）、受伤、使用道具 |
| 幸存者 | 待机、跟随、工作（建造/采集）、恐惧、受伤 |
| 丧尸 | 徘徊、追击、攻击、倒地、变异 |
| 佣兵 | 待机、战斗（各职业不同）、特殊技能释放 |

## 7. 音效与音乐方向

| 类别 | 方向 |
|------|------|
| 背景音乐 | 环境氛围音乐，低沉的弦乐 + 电子音色，动态切换 |
| 环境音效 | 风声、脚步回声、远处丧尸嘶吼、广播杂音 |
| UI 音效 | 简洁机械感，模拟手工操作反馈 |
| 战斗音效 | 枪声（不同武器差异化）、近战打击感、丧尸嘶吼 |
