# 「幸存者」游戏设计文档

> 版本：v0.1.0 | 日期：2026-05-26

---

## 1. 游戏世界观

### 1.1 背景设定

以丧尸危机爆发的末世为背景。主角需要在自身及伙伴的保护下共同对抗末世，为了寻找幸存者的踪迹，逐渐发现更大规模的丧尸世界的真相。

游戏营造一种荒废与混乱交织的氛围，在德国制造式末世理论的基调下，探寻废墟中的真实信息。整体叙事风格参考《这是我的战争》，在与 NPC 交流时以问句结尾，给玩家留下思考空间。

### 1.2 世界观关键词

| 关键词 | 说明 |
|--------|------|
| 丧尸末世 | 全球性丧尸危机爆发，文明秩序崩溃 |
| 幸存者联盟 | 主角需要寻找并团结其他幸存者 |
| 真相探索 | 逐渐揭开丧尸危机的深层原因 |
| 安全屋 | 幸存者们共同建设与守护的生存据点 |
| 变异进化 | 丧尸并非一成不变，存在变异与进化趋势 |

## 2. 剧情脚本

### 2.1 序章：从噩梦中醒来

故事从主角在自家床上醒来开始。窗外传来满天的灾难警报声，收音机里不断重播着紧急广播。主角走出家门，发现街道已陷入混乱——翻倒的车辆、散落的行李、墙壁上的血迹。

### 2.2 第一章：真相的碎片

在一处小巷里，主角发现一个奄奄一息的丧尸。出乎意料的是，这个丧尸尚存一丝残留意识，从它断断续续的话语中，主角得知了事件的开端线索，并获得了第一件武器。

丧尸临终前透露：城市周边的物流仓库中存有大量未被掠夺的物资库存。这个信息成为主角生存下去的初始动力。

### 2.3 第二章：安全屋的建立

主角下定决心前往物流仓库寻找物资，并在途中寻找适合建立安全屋的地点。在安全屋建设过程中，主角开始收集和改装各种特殊物品——这些道具不仅是生存工具，更是后期联系剧情的重要线索。

### 2.4 剧情推进机制

- **广播系统**：通过修复的收音机设备，接收外部世界的零星信息
- **道具线索**：收集的特殊物品在改装后逐渐揭示深层剧情
- **NPC 对话树**：与幸存者的对话推进支线剧情，多结局分支
- **日志系统**：主角的日记记录关键发现，串联剧情脉络

## 3. 核心玩法

### 3.1 游戏定位

**策略角色扮演类生存游戏（Strategy RPG Survival）**。玩家需要在资源稀缺的末世环境中，平衡探索、战斗、建设与幸存者管理，做出影响生存的关键决策。

### 3.2 核心循环

```
探索地图 → 获取物资/幸存者 → 建设基地 → 抵御丧尸 → 再次探索
```

### 3.3 核心机制：资源平衡

| 平衡维度 | 说明 |
|----------|------|
| 弹药管理 | 每次探索需权衡开枪次数，子弹是有限资源 |
| 食物分配 | 幸存者数量增长将消耗更多食物 |
| 幸存者决策 | 有限的幸存者需要分配探索/建设/生产任务 |
| 丧尸冲突周期 | 经过一定时间后，丧尸会主动进攻基地 |

### 3.4 探索收益

每局成功探索后，玩家将带回：
- 本次搜索到的物资（随机种类与数量）
- 可能发现的新幸存者
- 特殊物品/道具

以上收益用于基地建设与生存维持。

## 4. 随机探索机制

### 4.1 地图结构

探索地图由**箱组（Box Module）**组成，每次进入随机生成布局。

#### 小型箱

| 名称 | 功能说明 |
|------|----------|
| 随机小屋 | 可能产生「诅咒符文」——增益/减益效果 |
| 地产模块 | 产生住房单元，可能包含幸存者 |
| 商铺 | 产生商业类物资（食品、工具等） |
| 住宅 | 产生生活类物资及潜在幸存者 |

#### 大型箱

| 名称 | 功能说明 |
|------|----------|
| 大型商铺 | 大量商业物资，较高风险 |
| 大型电梯 | 通往特殊楼层，可能包含大型商铺或大型农田 |
| 大型农田 | 产生食物类资源及农业工具 |

### 4.2 房间类型

| 房间类型 | 描述 |
|----------|------|
| 幸存者房间 | 发现其他幸存者，可招募或互动 |
| 丧尸房间 | 遭遇丧尸，需要战斗或潜行 |
| 空房间 | 无特殊内容，可快速通过 |
| 休闲屋 | 恢复体力/理智的休息点 |
| 物资房间 | 主要资源获取来源 |
| 随机惊喜房间 | 触发特殊事件或获得稀有物品 |

### 4.3 变异感染者

- **普通丧尸**：需要 3 发子弹处决
- **变异人类**：仅需 1 发子弹处决，但数量稀少
- **特殊感染者**：具有独特机制，需要特定策略应对

## 5. 幸存者设计

### 5.1 出现机制

幸存者以较低几率随机出现在地图各个房间中，更倾向于分布在离入口较近的房间。

### 5.2 属性系统

每个幸存者拥有随机生成的属性：

| 属性 | 作用 |
|------|------|
| 力量 | 影响战斗能力、携带物资上限 |
| 意志 | 决定是否会主动探索房间；意志薄弱者不会主动离开所在房间（此类不超过总数 10%） |
| 敏捷 | 影响行动速度与建造速度的「理智」基础值 |
| 血量 | 决定幸存者是否会主动探索（血量过低时倾向于留守） |
| 健康值 | 健康值越高越有利于生存；可在救助站/医疗设施中恢复 |

### 5.3 行为逻辑

- **高意志幸存者**：会主动探索周边房间，帮助搜集物资
- **低意志幸存者**：只会对闯入者发起攻击，不会主动移动
- **血量影响**：血量不足时减少主动探索倾向
- **避难机制**：在丧尸袭击触发时，幸存者会自动寻找最近的安全区域

## 6. 佣兵设计

### 6.1 佣兵定位

佣兵是玩家的行动助力，不仅可以协助玩家一起探索地图，也可以独立派遣到其他地区搜集物资。

每个佣兵都拥有自己的背景故事及独立日常行为，有的倾向于建设领地，有的更擅长战场作战。

### 6.2 基础属性

每个佣兵拥有六项基础数据（评分 1~10 分）：

| 属性 | 含义 |
|------|------|
| 景气 | 影响团队士气加成 |
| 近战 | 近身战斗能力 |
| 品质 | 忠诚度与可靠性 |
| 武力 | 使用武器/远程攻击能力 |
| 智力 | 策略决策与特殊技能触发 |
| 体质 | 生命值与抗伤能力 |

### 6.3 佣兵等级

| 等级 | 点数范围 | 特征 |
|------|----------|------|
| S 级 | 20~40 分 | 独特背景故事，最高战力，但自负 |
| A 级 | 10~20 分 | 统一类型背景，中坚力量 |
| B 级 | 统一支援型 | 基础设施建设与辅助 |

### 6.4 职业方向

佣兵根据属性组合分为四种职业方向：

| 职业方向 | 擅长领域 |
|----------|----------|
| 卫生 | 基地维护、医疗、防疫 |
| 抢劫 | 物资搜集效率提升 |
| 勘探 | 探索范围扩大、发现隐藏内容 |
| 建设 | 基地建造速度与质量 |

### 6.5 S 级佣兵特殊机制

- **自负惩罚**：同一探险队伍中拥有多名 S 级佣兵时，会产生负面影响甚至罢工
- **天赋玩家（主角）**：拥有天赋的主玩家可自行开始探索行动，按自身天赋进行决策，不受 S 级佣兵自负影响
