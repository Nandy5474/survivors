## 当前版本功能（v0.2.0）

| 模块 | 功能 | 说明 |
|------|------|------|
| **场景系统** | 启动 → 菜单 → 游戏 流程 | 完整场景切换，暂停/恢复 |
| **HUD** | 生命/理智/天数/时间/资源显示 | 实时刷新，HTML 覆盖层 |
| **时间系统** | 游戏内时间推进 | 1 秒 = 1 分钟，跨天计算 |
| **暂停系统** | ESC 暂停，恢复/存档/退出 | 暂停遮罩层，事件正确解绑 |
| **存档系统（底层）** | IndexedDB 多槽位读写 | 5 个手动槽位 + 1 个自动槽位 |
| **配置系统** | LocalStorage 音量/语言等 | 默认值兜底，quota 异常处理 |
| **通知系统** | 屏幕顶部滑入通知 | 3 秒自动消失，info/warning/error 类型 |
