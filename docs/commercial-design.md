# 「幸存者」商业化设计文档

> 版本：v0.1.0 | 日期：2026-05-26

---

## 1. 商业化总览

### 1.1 商业模式定位

采用 **免费下载 + 内购（F2P + IAP）** 模式。核心设计原则：付费加速但不破坏平衡，不付费玩家可通过时间投入获得所有核心内容。

### 1.2 付费系统架构

```
┌─────────────────────────────────────┐
│           付费系统架构              │
├─────────────────────────────────────┤
│  一级货币（金币）← 游戏内产出      │
│  二级货币（钻石）← 充值获得        │
│  付费道具 ← 钻石购买               │
│  佣兵抽取 ← 钻石/付费券            │
│  角色获取 ← 付费解锁               │
└─────────────────────────────────────┘
```

## 2. 货币与道具系统

### 2.1 货币体系

| 货币 | 获取方式 | 用途 |
|------|----------|------|
| **金币** | 探索收益、基地生产、任务奖励 | 基础建设、基础物资购买、幸存者日常消耗 |
| **钻石** | 充值、活动奖励、成就奖励 | 佣兵抽取、高级道具、角色解锁、加速建造 |

### 2.2 付费道具分类

#### 生活及建设物资

| 道具类型 | 内容 | 定价策略 |
|----------|------|----------|
| 基础物资包 | 食物、水、基础建材 | 低单价，高频消耗 |
| 高级建设包 | 稀有建材、防御工事材料 | 中等单价 |
| 紧急补给包 | 弹药、医疗用品 | 中等单价，场景触发购买 |

#### 探索辅助道具

| 道具类型 | 效果 | 定价策略 |
|----------|------|----------|
| 生存率加成卡 | 单次探索中生存率 +X% | 低单价，按次消费 |
| 幸运符 | 提高稀有物资掉落概率 | 中等单价 |
| 雷达 | 揭示隐藏房间位置 | 中等单价 |
| 紧急撤离 | 探索失败时保留 50% 物资 | 高单价 |

#### 基础素材付费

| 内容 | 说明 |
|------|------|
| 木材/石材/金属包 | 用于基地建设的批量素材 |
| 种植包 | 种子、肥料等农业生产资料 |
| 改装零件 | 用于道具改装的零件套件 |

## 3. 佣兵抽取系统

### 3.1 抽取机制

| 项目 | 说明 |
|------|------|
| 抽取方式 | 钻石抽取 / 付费抽取券 |
| 单抽 | 消耗少量钻石，随机获得 B~S 级佣兵 |
| 十连抽 | 享受折扣，保底至少 1 个 A 级 |
| 保底机制 | N 次抽取未获得 S 级时，下一次必出 S 级 |

### 3.2 佣兵等级分布

| 等级 | 抽取概率 | 特征 |
|------|----------|------|
| S 级 | 约 3% | 20~40 总属性点，独特背景故事，专属技能 |
| A 级 | 约 22% | 10~20 总属性点，统一类型背景 |
| B 级 | 约 75% | 统一支援型，基础设施辅助 |

### 3.3 S 级佣兵独特价值

- 最高战力数值（20~40 属性点）
- 每个 S 级佣兵拥有完整的个人背景故事剧情线
- 独特的专属技能与战斗演出
- 自负机制增加策略深度（多名 S 级同队产生负面效果）

## 4. 角色获取机制

### 4.1 主角

主角根据背景故事进行获取，分为免费与付费两类：

| 主角类型 | 获取方式 |
|----------|----------|
| 初始主角 | 免费，游戏开局赠送，完整主线剧情 |
| 额外主角 | 付费解锁，拥有不同的背景故事与天赋技能 |

### 4.2 S 级角色

部分 S 级角色通过以下方式获取：

| 获取方式 | 说明 |
|----------|------|
| 免费获取 | 完成特定主线/支线剧情解锁 |
| 活动赠送 | 限时活动中的奖励角色 |
| 付费解锁 | 直接购买或通过高级通行证获取 |

### 4.3 角色获取一览

| 获取方式 | 免费 | 付费 |
|----------|------|------|
| 主线剧情解锁 | 初始主角 + 若干关键 NPC | - |
| 支线达成奖励 | 部分 A/S 级角色 | - |
| 佣兵抽取 | B/A 级佣兵 | S 级佣兵（概率） |
| 直接购买 | - | 限定主角、限定 S 级角色 |
| 通行证 | 免费轨道奖励 | 付费轨道专属角色 |

## 5. 付费点总览

### 5.1 核心付费项

| 付费项 | 类型 | 是否影响平衡 |
|--------|------|--------------|
| 钻石充值 | 货币 | 加速型 |
| 佣兵十连抽 | 抽卡 | 收集型 |
| 限定角色购买 | 直购 | 内容型 |
| 高级建设礼包 | 消耗品 | 加速型 |
| 探索辅助道具 | 消耗品 | 辅助型 |
| 通行证 | 订阅 | 混合型 |

### 5.2 非付费路径保障

| 内容 | 免费获取方式 |
|------|--------------|
| 金币 | 探索、生产、任务（无限产出） |
| 钻石 | 成就、签到、活动（有限产出） |
| B 级佣兵 | 探索中招募、任务奖励 |
| A 级佣兵 | 抽取保底、活动赠送 |
| 基础物资 | 探索产出、基地生产 |
| 主线剧情 | 完全免费 |
