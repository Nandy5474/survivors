## 风险与依赖

| 风险 | 可能性 | 影响 | 缓解措施 |
|------|--------|------|----------|
| IndexedDB 在隐私模式不可用 | 中 | 存档失效 | LocalStorage 兜底方案 |
| Canvas 性能在低端机不足 | 高 | 帧率下降 | 动态降低分辨率/特效 |
| 美术资源交付延期 | 中 | 内容不足 | 使用占位方块先行开发 |
| Web Audio API 兼容性问题 | 低 | 音效缺失 | 静默降级 |
| GitHub Pages 带宽限制 | 低 | 部署失败 | 备选 CDN |
